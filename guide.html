Freelancer Auto Bid Assistant - Help

Introduction

The Freelancer Auto Bid Assistant helps you automate your bids on Freelancer by collecting project data and generating personalized bid responses. With this extension, you can manage your project list, update settings, and use AI-powered tools to craft perfect bids with minimal effort.

How to Use

Step 1: Accessing the Popup

Click the extension icon in your browser toolbar to open the popup. The popup will display a list of current projects along with options to filter and refresh the project list.

Step 2: Refreshing the Project List

To update the project list, simply click the refresh button. An animation will indicate that the list is being updated, and the refreshed projects will be displayed once the update is complete.

Step 3: Viewing Projects

Each project is shown with its title, price, snippet, and summary. Click on a project to view more details, and you can enable auto-bidding for it based on your settings.

Settings Overview

Profile Information

In the settings page, you can configure your personal details, including your name, title/role, experience summary, and skills. These details will be used by the extension to personalize the bids.

App Preferences

In the "App Preferences" section, you can adjust the following:

  • Fetch Interval: Set how often the extension will fetch new project data (in minutes).
  • AI Integration Key: Enter your OpenAI API key to enable AI-powered bid generation.
  • Auto-Fill Bids: Toggle this option to automatically fill bids based on your profile information.
  • Notifications: Enable or disable notifications for new project updates.
  • Notifications Auto-Disappear: Choose whether notifications should automatically disappear after a certain time.

Troubleshooting

Extension Not Updating Projects

If you notice that the project list is not updating properly, try refreshing the page or adjusting your fetch interval settings.

AI Bidding Not Working

Ensure that your API key is correctly entered in the settings page. If the problem persists, verify your API key with the service provider.
